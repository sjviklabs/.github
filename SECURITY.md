# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any SJVIK Labs repository, please report it responsibly.

**Email:** security@stevenjvik.tech

Please include:
- Description of the vulnerability
- Steps to reproduce
- Affected repository and version (if applicable)

We aim to acknowledge reports within 48 hours and provide a fix or mitigation plan within 7 days.

## Supported Versions

Only the latest version of each project on the default branch is supported with security updates.

## Scope

This policy applies to all public repositories under the [sjviklabs](https://github.com/sjviklabs) organization.
